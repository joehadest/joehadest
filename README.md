### 🚀 Projetos em Destaque

#### 🌐 Webpulse
Uma landing page profissional focada em transformar ideias em experiências digitais.
* **Tech Stack:** Next.js, React, TypeScript, Tailwind CSS.
* **Destaques:** Componentização avançada, animações de UI e interface totalmente responsiva.

#### ⚙️ Shellton Auto Mecânica
Site e painel administrativo completo para oficina mecânica. Solução voltada para o dia a dia prático do negócio, incluindo portfólio de serviços, sistema de agendamento online, fila pública de atendimento em tempo real e dashboard admin.
* **Tech Stack:** Next.js, TypeScript, Tailwind CSS e Supabase Realtime.
* 🔗 [Visitar site](https://shelltonautomecanica.webpulseservicos.com/) 

#### 👕 Seu Manto
E-commerce full-stack de camisetas. Aplicação robusta que conta com loja virtual, carrinho de compras, fluxo de checkout completo e painel administrativo.
* **Tech Stack:** Supabase Realtime e Integração com Mercado Pago.
* 🔗 [Visitar site](https://seumanto.webpulseservicos.com/) 

#### 🍔 Rei dos Salgados
App completo de delivery, desenhado para otimizar o fluxo de pedidos. Possui carrinho de compras, rastreamento de pedidos e um painel de controle dedicado para a organização da cozinha.
* **Tech Stack:** Next.js e MongoDB.
* 🔗 [Visitar site](https://reidossalgados.webpulseservicos.com/) 

#### 🎫 Camarote PMD
Landing page dinâmica de eventos focada na conversão e venda de ingressos, apresentando o line-up completo e galeria de fotos.
* **Tech Stack:** React, Vite e animações.
* 🔗 [Visitar site](https://camarotepmd.com.br/) 

#### 🏭 Baracho Industrial
Site institucional criado para o setor de engenharia, destacando áreas de atuação rigorosas como alpinismo industrial, solda e manutenção.
* **Tech Stack:** React e Tailwind CSS.
* 🔗 [Visitar site](https://www.barachosolucoes.com/) 

#### 🏗️ DRM Serviços
Site institucional desenvolvido para serviços industriais. Possui gerenciamento dinâmico com seções editáveis de serviços, exibição de projetos em destaque e área facilitada para contatos comerciais.
* 🔗 [Visitar site](https://drmm.webpulseservicos.com/)
